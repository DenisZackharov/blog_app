# Blog App - React Frontend

React приложение для блога с аутентификацией, управлением постами и комментариями. Работает с бэкендом на FastAPI.

## Возможности

- **Аутентификация**: Регистрация, Вход, Выход, Управление профилем
- **Посты**: Создание, Просмотр, Редактирование, Удаление
- **Комментарии**: Добавление, Редактирование, Удаление
- **Управление правами**:
  - Создание постов — только для авторизованных
  - Редактирование/удаление поста — только для автора
  - Комментарии — форма только для авторизованных
  - Редактирование/удаление комментария — только для автора комментария

## Установка

### Требования

- Node.js >= 16
- npm или yarn

### Шаги

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

## Режим работы без бэкенда (Mock Data)

Приложение **может работать без бэкенда** с использованием mock-данных.

### Как включить:

Откройте файл `.env` и установите:
```env
VITE_USE_MOCK=true
```

### Тестовые учетные данные:

| Username | Password | Email |
|----------|----------|-------|
| admin | admin123 | admin@example.com |
| user1 | user1234 | user1@example.com |

### Как отключить (режим реального бэкенда):

```env
VITE_USE_MOCK=false
VITE_API_URL=http://localhost:8000/api/v1
```

## Настройка подключения к реальному API

Создайте файл `.env` в корне проекта:

```env
VITE_USE_MOCK=false
VITE_API_URL=http://your-api-url/api/v1
```

## API Endpoints

Приложение ожидает следующие эндпоинты на бэкенде (префикс `/api/v1`):

### Аутентификация

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/v1/auth/register` | Регистрация пользователя |
| POST | `/api/v1/auth/login` | Вход в систему |
| POST | `/api/v1/auth/logout` | Выход из системы |
| GET | `/api/v1/auth/me` | Получение данных текущего пользователя |
| PUT | `/api/v1/auth/profile` | Обновление профиля |

### Посты

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/posts` | Список всех постов |
| GET | `/api/v1/posts/:id` | Получение поста по ID |
| POST | `/api/v1/posts` | Создание поста |
| PUT | `/api/v1/posts/:id` | Обновление поста |
| DELETE | `/api/v1/posts/:id` | Удаление поста |

### Комментарии

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/v1/posts/:id/comments` | Комментарии к посту |
| POST | `/api/v1/posts/:id/comments` | Создание комментария |
| PUT | `/api/v1/posts/:id/comments/:commentId` | Обновление комментария |
| DELETE | `/api/v1/posts/:id/comments/:commentId` | Удаление комментария |
