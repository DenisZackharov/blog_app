# Blog App - React Frontend

React приложение для блога с аутентификацией, управлением постами и комментариями. Работает с бэкендом на FastAPI.

## Возможности

- **Аутентификация**: Регистрация, Вход, Выход, Управление профилем
- **Посты**: Создание, Просмотр, Редактирование, Удаление
- **Комментарии**: Добавление, Редактирование, Удаление
- **Управление правами**:
  - Создание постов — только для авторизованных
  - Редактирование/удаление поста — только для автора
  - Комментарии — форма только для авторизованных
  - Редактирование/удаление комментария — только для автора комментария

## Установка

### Требования

- Node.js >= 16
- npm или yarn

### Шаги

1. Клонируйте репозиторий (или перейдите в директорию проекта)

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

## Настройка подключения к API

### По умолчанию

Приложение настроено на подключение к бэкенду по адресу `http://localhost:8000/api`.

### Через переменные окружения

Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://your-api-url/api
```

### Прокси

Vite настроен с прокси для всех запросов к `/api` на бэкенд `http://localhost:8000`. Это настроено в [`vite.config.js`](vite.config.js).

## API Endpoints

Приложение ожидает следующие эндпоинты на бэкенде:

### Аутентификация

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация пользователя |
| POST | `/api/auth/login` | Вход в систему |
| POST | `/api/auth/logout` | Выход из системы |
| GET | `/api/auth/me` | Получение данных текущего пользователя |
| PUT | `/api/auth/profile` | Обновление профиля |

### Посты

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/posts` | Список всех постов |
| GET | `/api/posts/:id` | Получение поста по ID |
| POST | `/api/posts` | Создание поста |
| PUT | `/api/posts/:id` | Обновление поста |
| DELETE | `/api/posts/:id` | Удаление поста |

### Комментарии

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/posts/:id/comments` | Комментарии к посту |
| POST | `/api/posts/:id/comments` | Создание комментария |
| PUT | `/api/posts/:id/comments/:commentId` | Обновление комментария |
| DELETE | `/api/posts/:id/comments/:commentId` | Удаление комментария |
